# Specification: 优化 AI 提示词与上下文注入

## 目标 (Objective)
提高 DeepCurrents 每日宏观研报的质量、逻辑深度和可读性。确保 AI 引擎能够更准确地从碎片化的全球资讯中提取关键趋势，并提供具有实战价值的资产研判。

## 范围 (Scope)
- **核心逻辑优化**: 修改 `src/services/ai.service.ts` 或相关的研报生成模块，改进 Prompt 模板。
- **上下文增强**: 优化聚类事件、趋势关键词及其权重的注入方式。
- **测试覆盖**: 为新的 Prompt 模板和生成逻辑编写单元测试与集成测试。

## 验收标准 (Acceptance Criteria)
1. **内容逻辑性**: 生成的研报需包含清晰的资产类别研判（看涨/看跌）、关键宏观趋势分析及核心事件摘要。
2. **格式正确性**: 严格遵循预定义的输出格式（如 Markdown 或飞书/Telegram 卡片格式）。
3. **性能指标**: 优化后的上下文注入逻辑需满足 Token 预算管理要求，避免由于上下文过长导致的 API 失败。
4. **代码质量**: 模块测试覆盖率需达到 >80%，无 Lint 错误。

## 非功能性需求
- **灵活性**: 支持通过环境变量或外部配置轻松调整 Prompt 模板。
- **可观测性**: 记录详细的 Prompt 生成日志，便于后续调试和效果对比。
